Project Overview

This website was developed as part of the Operating System course to serve as a structured guide for performing system administration tasks using Oracle VirtualBox. It documents hands-on activities, installations, and configurations commonly used in enterprise environments, allowing users to follow along and understand each process in a clear and organized way.

Scope of Activities

  • Windows Server Installation
  • Active Directory Configuration
  • DHCP Role Setup & Workstations in Domain
  • RAID Installation
  • Group Policy Management
  • File Services Configuration
  • Various Windows OS Installations

Case Studies

The project also includes a series of case studies focused on real-world server administration scenarios. These activities simulate enterprise environments using a domain named tupt.edu.ph, demonstrating how systems are deployed, managed, and secured.

  • Domain Controller Setup — tupt.edu.ph
  • Adding and Managing Users
  • DHCP Server Configuration
  • File Sharing Implementation
  • RAID Configuration
  • Remote Desktop Setup
  • Web Server Deployment
  • Group Policy Configuration
  • Windows Deployment Services

Purpose

The purpose of this website is to simplify complex system administration tasks by presenting them in a structured and easy-to-follow format. It acts as both a learning reference and a practical guide for students working with virtualization and Windows Server environments.
