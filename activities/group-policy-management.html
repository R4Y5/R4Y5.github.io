GROUP

POLICIES.

Description

Configure Group Policy Objects (GPOs) to enforce system restrictions and security policies across the network environment.

Implemented Policies

  • Remove Windows Update access
  • Remove Logoff option from Start Menu
  • Disable Task Manager — CTRL + ALT + DEL
  • Customize Internet Explorer Title Bar
  • Restrict LAN Connection Properties
  • Prevent Access
    • Command Prompt
    • Registry Editor
    • Control Panel
  • Access Control Policy
    • Administrators retain full system control
    • Standard users are restricted
