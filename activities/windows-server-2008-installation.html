Description

Windows Server 2008 Installation Guide using Oracle VirtualBox. This guide walks through creating and configuring a virtual machine for educational and lab purposes.

Requirements

  • Software
    • Install Oracle VirtualBox
    • Install VirtualBox Extension Pack (matching version)
  • Windows Server 2008 ISO
    • Microsoft evaluation downloads
    • Academic software portals
    • School-provided installer
  • Virtual Machine Specifications
    • RAM: 2 GB
    • CPU: 2 cores
    • Storage: 40 GB
    • Graphics: 128 MB video memory

Steps

  • Create Virtual Machine
    • Open VirtualBox → Click New
    • Name: Windows Server 2008
    • Type: Microsoft Windows
    • Version: Windows 2008 (64-bit)
    • Click Next
  • Allocate Memory
    • Set RAM to 2048 MB (2 GB)
    • Click Next
  • Create Virtual Hard Disk
    • Select "Create a virtual hard disk now"
    • Choose VDI format
    • Select Dynamically Allocated
    • Set size to 40 GB
    • Click Create
  • Attach Windows Server ISO
    • Select VM → Settings → Storage
    • Under Controller IDE, select Empty
    • Click disk icon → Choose disk file
    • Select Windows Server 2008 ISO
    • Click OK
  • Start Installation
    • Click Start
    • Select language and time format
    • Click Next
  • Install Windows
    • Click Install Now
    • Select edition (Standard / Enterprise)
    • Accept license terms
    • Choose Custom (Advanced)
  • Partition Disk
    • Select virtual disk
    • Click New → Create partition
    • Click Next
    • Wait for installation and restarts
  • Administrator Setup
    • Create Administrator password
    • Log in to the server
  • Install Guest Additions
    • Click Devices → Insert Guest Additions CD
    • Run VBoxWindowsAdditions.exe
    • Restart VM
    • Improves graphics and performance
  • Activation Options
    • Use valid Microsoft product key
    • Use school/enterprise license
    • Use evaluation license (limited)
    • Activate via Control Panel → System
