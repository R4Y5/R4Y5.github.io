ACTIVITY 6 :

FILE

SERVICES.

Description

Configure File Services in Windows Server to create a centralized storage system for domain users. This setup enables secure file sharing, access control, and efficient management of shared resources across the network.

Tasks

  • Install File Services Role
  • Create shared folders with permissions and file restrictions
  • Manage access control for different user groups

Shared Folders

  • Public Folder
    • Permissions: Read / Write
    • Users: All
    • Allowed Files: All except .exe, .com, .vb, .vbs
  • VB.Net Folder
    • Permissions: Read / Write
    • Allowed Files: .zip, .rar
  • C_Programs Folder
    • Permissions: Read / Write
    • Allowed Files: .c, .cpp
  • Surname_Section Folder
    • Permissions: Full Control
    • Users: Domain Users
    • Allowed Files: .c, .cpp, .pdf, .rar, .docx, .ppt, .dwg
  • PHOTO_Male & PHOTO_FEMALE
    • Permissions: Read / Write
    • Access: Based on user gender
    • Allowed Files: .jpg, .gif, .png
