Description

Configure Active Directory Domain Services (AD DS) and create user accounts for BSIT students within the domain environment. This setup enables centralized authentication, user management, and access control.

Tasks

  • Create user accounts for BSIT students — 1st to 4th year
  • Assign default password — p@ss123
  • Organize users by section using groups
  • Domain: tup.local

User Configuration

  • Student Accounts
    • Create accounts in Active Directory Users & Computers
    • Use consistent naming convention — e.g., lastname.firstname
    • Assign users to their respective section groups
    • Set initial password and require change on first login (optional)
  • Administrator Account
    • User cannot change password
    • Password never expires
    • Maintain full administrative privileges
  • Group Organization
    • Create groups per year/section — e.g., BSIT-1A, BSIT-2B
    • Assign permissions based on group membership
    • Simplifies management of shared resources

Best Practices

  • Avoid using the same password long-term for all users
  • Enable password policies via Group Policy
  • Use Organizational Units (OUs) for better structure
  • Regularly review and disable inactive accounts
