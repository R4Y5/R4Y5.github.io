ACTIVITY 3 :

DCHP

ROLE.

Description

Install and configure the DHCP Server role to automate IP address assignment and ensure proper domain connectivity for client machines. This setup supports structured network segmentation and centralized management of Windows XP and Windows 7 workstations.

Configuration

  • Role Installed — DHCP Server
  • Server IP — Last usable IP address
  • Lease Duration — 16 hours
  • Router — Server IP Address
  • DNS — Server IP + 8.8.8.8

Network Scopes

  • Class A Network — 10.0.0.0 /8 (Male students)
    • Exclusion Range: First 10 and last 10 usable IPs
    • Used for Windows XP clients
  • Class B Network — 172.16.0.0 /16 (Female students)
    • Exclusion Range: First 10 and last 10 usable IPs
    • Used for Windows 7 clients

Client Systems

  • Windows XP — LASTNAME-XP
  • Windows 7 — LASTNAME-WIN7

Requirements

  • IP Assignment
    • Windows XP must receive Class A IP
    • Windows 7 must receive Class B IP
  • Domain Integration
    • All clients must join the domain
    • Verify connectivity using domain login
  • User Testing
    • Login at least 5 student accounts per section
    • Test authentication and network access

Best Practices

  • Use reservations for critical devices (servers, printers)
  • Avoid overlapping IP ranges between scopes
  • Ensure DHCP service is authorized in Active Directory
  • Verify DNS resolution for domain services
