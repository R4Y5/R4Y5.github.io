CASE STUDY 6

REMOTE

DESKTOP.

Description

Configure Remote Desktop Services for remote administration and secure access. Design backup schedules and verify recovery procedures.

Prerequisites

  • Start Windows XP VM
  • Start Windows 7 VM
  • Start Windows Server 2008 VM

Steps

  • Enable Remote Desktop on Windows 7
    • Open Active Directory Users & Computers (Windows Server 2008)
    • Navigate to Computers container
    • Locate Windows 7 computer object
    • Right-click → Properties
    • Go to Remote Settings tab
    • Select: Allow connections from any version of Remote Desktop
    • Click Apply → OK
  • Connect to Windows 7 from Windows Server 2008
    • Open Remote Desktop Connection
    • Enter Windows 7 hostname
    • Click Connect
    • Enter user credentials
    • Accept connection prompt on Windows 7
    • Remote session established
  • Enable Remote Desktop on Windows XP
    • Right-click My Computer → Properties
    • Go to Remote tab
    • Check: Allow users to connect remotely
    • Click OK
  • Connect to Windows XP from Windows Server 2008
    • Log off Windows XP first
    • Open Remote Desktop Connection
    • Enter Windows XP hostname
    • Click Connect
    • Accept prompt on Windows XP
    • Enter user credentials
    • Remote session established
  • Important Reminder
    • Always log off Windows XP before connecting remotely
    • Same user cannot be logged in locally and remotely at the same time
