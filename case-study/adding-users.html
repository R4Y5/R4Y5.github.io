Steps

  • Create domain users for each department/office and technology section — use real names when creating domain accounts
  • Create an organizational unit for each technology section or office — e.g., computer engineering, electronics, registrar, research
  • Restrict all users to log on only to assigned workstations — XP class A network, 7 class B network
    • Users → Properties → Computer Restrictions → select which computer
  • Set password expiry to every 30 days for all users
    • Go to Group Policy Management → create a GPO for Academic & Administrative
    • Academic: Academic Password | Administrative: Administrative Password
    • Click Edit → Computer Config → Security Settings → Account Policies → Password Policies
    • Maximum Password Age → Properties → 30 days → Apply → OK
  • Set logon hour restrictions per user type
    • Academic users — 7:30 AM to 8:45 AM
    • Administrative staff — 7:00 AM to 7:00 PM
    • Select Users in each Department → Properties → Account → Log on Hours → select time range → log on permitted
